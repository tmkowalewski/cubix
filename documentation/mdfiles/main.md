Toolkit for Nuclei (TkN) is a C++ library providing an easy and fast access to a large variety of nuclear data measurements. 
It allows for example to access to ground state properties, level excited states energies and decay characteristics.

TkN can be used linked with ROOT, allowing to explore the TkN database interactively in the ROOT prompt terminal. 
It can also be compiled without any dependencies for lighter integration in simulation codes or theoretical nuclear models.

The source code is available on [GitLab](https://gitlab.in2p3.fr/tkn/tkn-lib).

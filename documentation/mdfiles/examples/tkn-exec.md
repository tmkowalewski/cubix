## Compile a simple program using TkN

`tkn-exec.cpp` provides a simple example program using TkN to print the ground state spin parity of the Carbon isotopic chain.

To compile this example, use :

```shell
g++ tkn-exec.cpp -o tkn-exec `tkn-config --cflags --linklibs`
```
